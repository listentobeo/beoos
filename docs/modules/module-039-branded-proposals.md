# Module 3.9: Branded Proposals and Catalogue-Powered Quotes

Module 3.9 makes quotations feel like client-ready proposals instead of internal calculations.

It builds on Module 3.8 by improving the public quote experience and connecting flexible quotes to the tenant price catalogue.

## Added in this module

- Branded public proposal pages.
- Template accent color support on the public quote header.
- Client-facing itemized quote tables.
- Print / save-as-PDF button on public proposals.
- Price catalogue picker inside the flexible quote creator.
- Add blank quote rows or add catalogue-backed rows.

## Why this matters

Businesses can now build quotes from their stored services/products and share a cleaner client-facing proposal link.

For Beo Art Studio, the mural calculator still works. For other businesses, the same quotation section can now support:

- art supplies bulk purchases;
- service packages;
- product bundles;
- retainers;
- installation jobs;
- inventory-backed quotes.

## What remains

The next layer should add:

- true server-generated PDF files;
- logo upload and storage;
- visual layout/theme selection;
- drag-and-drop quote item editing;
- AI quote assistant for creating/updating quotes from natural language.

The current implementation keeps the system fast by using browser print-to-PDF while preserving the structure needed for generated PDFs later.
